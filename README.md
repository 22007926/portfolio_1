 Project Overview

This project is a basic personal portfolio-style website consisting of multiple HTML pages connected together with a single CSS stylesheet.

The website includes:

    Homepage (index.html)

    About Me page (aboutme.html)

    Contact Me page (contactme.html)

    Projects overview page (projects.html)

    Individual project pages (project1.html, project2.html)



 How to Run the Website

    Extract the zip file (web div1.zip) if you haven't already.

    Open the folder named web div1.

    Locate and open index.html in your preferred web browser (e.g., Chrome, Firefox, Edge),or by use of visual studio

    From there, you can navigate through the website using the internal links.
