Java Calculator

This project is a basic calculator application developed in Java.

The application allows users to perform addition, subtraction, multiplication, and division. It utilizes a switch statement for operation selection.


Click--MY Java Calculator --To View Code on GitHub

The calculator is designed to be user-friendly, providing a simple interface for input and output. Users can enter two numbers and select the desired operation. The result is displayed immediately after the calculation.

Tools Used: Java, Visual Studio code

Note: Ensure you have Java Development Kit (JDK) installed to run this application.
