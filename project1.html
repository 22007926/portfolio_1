C++ Calculator

This project is a simple calculator program implemented in C++.


Click--MY C++ Calculator --To View Code on GitHub

This calculator allows users to perform basic arithmetic operations: addition, subtraction, multiplication, and division. It uses a switch statement to determine the operation to perform.

The program prompts the user to enter two numbers and the desired operation. After performing the calculation, it displays the result.

Tools Used: C++, Visual Studio Code

Note: Ensure you have a C++ compiler installed to run this application.
